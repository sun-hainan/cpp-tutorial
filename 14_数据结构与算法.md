## 1. 链表：节点的"手拉手"

### 生活类比

> **链表**就像小朋友**手拉手做游戏**——每个人（节点）只知道自己的数据和下一个小朋友的"手"（指针），要找到特定小朋友需要从第一个开始一个个数。

### 概念讲解

```cpp
template<typename T>
struct ListNode {
    T data;
    ListNode* next;
    ListNode(const T& d) : data(d), next(nullptr) { }
};

template<typename T>
class LinkedList {
public:
    void push_front(const T& val) {  // O(1)
        ListNode<T>* node = new ListNode<T>(val);
        node->next = head_;
        head_ = node;
        ++size_;
    }

    void push_back(const T& val) {  // O(n)
        ListNode<T>* node = new ListNode<T>(val);
        if (!head_) { head_ = node; }
        else {
            ListNode<T>* cur = head_;
            while (cur->next) cur = cur->next;
            cur->next = node;
        }
        ++size_;
    }

    bool find(const T& val) const {  // O(n)
        ListNode<T>* cur = head_;
        while (cur) {
            if (cur->data == val) return true;
            cur = cur->next;
        }
        return false;
    }

    void reverse() {  // O(n)
        ListNode<T>* prev = nullptr;
        ListNode<T>* cur = head_;
        while (cur) {
            ListNode<T>* next = cur->next;
            cur->next = prev;
            prev = cur;
            cur = next;
        }
        head_ = prev;
    }
};
```

### 复杂度

| 操作 | 时间复杂度 |
|------|-----------|
| push_front | O(1) |
| push_back | O(n) |
| find | O(n) |
| reverse | O(n) |

## 2. 栈与队列

### 生活类比

> **栈**像**洗碗的流程**——先洗的碗放最底下，后洗的碗叠在上面，用的时候从最上面拿（LIFO）。
> **队列**像**排队买票**——先来的先买，后来的排到最后（FIFO）。

### 概念讲解

```cpp
// 栈（LIFO）
std::stack<int> s;
s.push(1); s.push(2); s.push(3);
s.top();   // 3
s.pop();   // 移除 3

// 队列（FIFO）
std::queue<int> q;
q.push(1); q.push(2); q.push(3);
q.front();  // 1
q.pop();    // 移除 1

// 优先队列（堆）
std::priority_queue<int> pq;  // 大顶堆
pq.push(5); pq.push(1); pq.push(3);
pq.top();  // 5

// 小顶堆
std::priority_queue<int, std::vector<int>, std::greater<int>> min_pq;
min_pq.push(5); min_pq.push(1); min_pq.push(3);
min_pq.top();  // 1
```

### 栈应用：括号匹配

```cpp
auto is_balanced = [](const std::string& str) -> bool {
    std::stack<char> st;
    for (char c : str) {
        if (c == '(' || c == '[' || c == '{') st.push(c);
        else if (c == ')' && (st.empty() || st.top() != '(')) return false;
        else if (c == ']' && (st.empty() || st.top() != '[')) return false;
        else if (c == '}' && (st.empty() || st.top() != '{')) return false;
        else if (")]}".find(c) != std::string::npos) st.pop();
    }
    return st.empty();
};
```

## 3. 哈希表

### 生活类比

> **哈希表**就像图书馆的**检索系统**——每本书有一个编号（哈希值），查找时直接翻到对应编号的柜子，而不是按字母顺序一页页翻。

### 概念讲解

```cpp
std::unordered_map<std::string, int> dict = {
    {"apple", 1}, {"banana", 2}, {"cherry", 3}
};

// 插入和查找
dict["date"] = 4;
auto it = dict.find("banana");
if (it != dict.end()) {
    std::cout << "banana = " << it->second << std::endl;
}

// 哈希冲突：链地址法
// 负载因子 > max_load_factor 时 rehash
std::cout << "bucket_count = " << dict.bucket_count() << std::endl;
```

## 4. 二叉树遍历

### 生活类比

> **二叉树**像一棵**倒过来的树**——根在上，叶在下。每个节点最多有两个"孩子"。

### 四种遍历

```
       1
      / \
     2   3
    / \   \
   4   5   6

前序（根-左-右）：1 2 4 5 3 6
中序（左-根-右）：4 2 5 1 3 6
后序（左-右-根）：4 5 2 6 3 1
层序（队列）：1 2 3 4 5 6
```

### 代码示例

```cpp
// 前序遍历
auto preorder = [](auto&& self, TreeNode* node) -> void {
    if (!node) return;
    std::cout << node->val << " ";
    self(self, node->left);
    self(self, node->right);
};

// 中序遍历
auto inorder = [](auto&& self, TreeNode* node) -> void {
    if (!node) return;
    self(self, node->left);
    std::cout << node->val << " ";
    self(self, node->right);
};

// 后序遍历
auto postorder = [](auto&& self, TreeNode* node) -> void {
    if (!node) return;
    self(self, node->left);
    self(self, node->right);
    std::cout << node->val << " ";
};

// 层序遍历（队列）
auto levelorder = [](TreeNode* root) {
    std::queue<TreeNode*> q;
    q.push(root);
    while (!q.empty()) {
        TreeNode* node = q.front(); q.pop();
        std::cout << node->val << " ";
        if (node->left) q.push(node->left);
        if (node->right) q.push(node->right);
    }
};
```

## 5. 红黑树与 std::map

### 红黑树性质

```
1. 节点非红即黑
2. 根节点是黑色
3. 叶子节点（NIL）是黑色
4. 红节点的子节点必须是黑色
5. 从任一节点到叶子的路径黑高相同
→ 最长路径 ≤ 2 × 最短路径（近似平衡）
```

### std::map 特性

| 操作 | 复杂度 |
|------|--------|
| 插入 | O(log n) |
| 删除 | O(log n) |
| 查找 | O(log n) |
| 遍历 | 按键有序 |

### map vs unordered_map

| 特性 | map | unordered_map |
|------|-----|---------------|
| 查找 | O(log n) | O(1) 平均 |
| 有序遍历 | ✅ | ❌ |
| 稳定迭代器 | ✅ | 可能失效 |

## 6. 排序算法

### 算法对比表

| 算法 | 时间复杂度 | 空间复杂度 | 稳定性 |
|------|-----------|-----------|--------|
| 冒泡排序 | O(n²) | O(1) | 稳定 |
| 选择排序 | O(n²) | O(1) | 不稳定 |
| 插入排序 | O(n²) | O(1) | 稳定 |
| 归并排序 | O(n log n) | O(n) | 稳定 |
| 快速排序 | O(n log n) 均摊 | O(log n) | 不稳定 |
| 堆排序 | O(n log n) | O(1) | 不稳定 |

### 代码示例

```cpp
// std::sort = introsort（快排+堆排+插入排的混合）
std::vector<int> test = {5, 2, 8, 1, 9, 3};
std::sort(test.begin(), test.end());
// 降序
std::sort(test.begin(), test.end(), std::greater<int>());
```

## 7. 查找算法

### 复杂度对比

| 算法 | 时间复杂度 | 数据要求 |
|------|-----------|---------|
| 顺序查找 | O(n) | 无 |
| 二分查找 | O(log n) | 有序数组 |
| 哈希查找 | O(1) 平均 | 哈希表 |
| BST 查找 | O(log n) | 二叉搜索树 |

### 二分查找代码

```cpp
auto binary_search = [](const std::vector<int>& arr, int target) -> int {
    int left = 0, right = static_cast<int>(arr.size()) - 1;
    while (left <= right) {
        int mid = left + (right - left) / 2;  // 防止溢出
        if (arr[mid] == target) return mid;
        else if (arr[mid] < target) left = mid + 1;
        else right = mid - 1;
    }
    return -1;
};
```

## 本章小结

| 数据结构 | 核心特点 | 适用场景 |
|---------|---------|---------|
| 链表 | O(1) 插入/删除，O(n) 查找 | 频繁中间操作 |
| 栈 | LIFO | 括号匹配、函数调用 |
| 队列 | FIFO | 任务调度、广度优先 |
| 优先队列 | 大顶堆/小顶堆 | TopK、中位数 |
| 哈希表 | O(1) 平均查找 | 高性能键值查找 |
| 二叉树 | 层次结构 | 文件系统、表达式树 |
| 红黑树 | O(log n) 近似平衡 | std::map/std::set |
