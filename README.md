## 📖 模块格式

每个模块包含：
- **学习目标**：本章核心知识点
- **生活类比**：用生活场景理解抽象概念
- **概念讲解**：原理和背景知识
- **代码示例**：逐行注释的完整代码
- **ASCII 图示**：可视化数据结构和工作原理
- **常见错误**：踩坑提醒
- **本章小结**：核心要点速查表
- **课后练习**：5 道实践题

## 📝 学习建议

1. **先读后练**：先理解原理，再完成练习
2. **对比学习**：每个模块包含与其他语言（Python/Rust/Go）的对比
3. **动手实验**：修改代码，观察运行结果
4. **循序渐进**：不建议跳跃式学习
5. **复习巩固**：每学完几个模块，回顾前面的内容
