## 1. if/else：程序的"十字路口"

### 生活类比

> if/else 就像**十字路口的交通灯**——条件为 true 走绿灯分支，false 走红灯分支。

### 概念讲解

```cpp
// 基本 if 语句
int score = 85;
if (score >= 60) {
    std::cout << "及格" << std::endl;
} else {
    std::cout << "不及格" << std::endl;
}

// else-if 链（多条件分支）
if (score >= 90) {
    std::cout << "优秀" << std::endl;
} else if (score >= 80) {
    std::cout << "良好" << std::endl;
} else if (score >= 70) {
    std::cout << "中等" << std::endl;
} else if (score >= 60) {
    std::cout << "及格" << std::endl;
} else {
    std::cout << "不及格" << std::endl;
}
```

### C++17 if with initializer

```cpp
// if 语句支持初始化（作用域限于 if 内）
std::map<std::string, int> ages = {{"Alice", 25}, {"Bob", 30}};
if (auto it = ages.find("Alice"); it != ages.end()) {
    std::cout << "Alice's age = " << it->second << std::endl;
}
```

### 常见陷阱

```cpp
// 陷阱1：悬空 else（else 绑定到最近的 if）
int x = 0;
if (x > 0)
    if (x > 10)
        std::cout << "x > 10" << std::endl;
    else
        std::cout << "x is 0~10" << std::endl;  // ⚠️ else 绑定到内层 if！

// ✅ 正确写法：用花括号
if (x > 0) {
    if (x > 10) {
        std::cout << "x > 10" << std::endl;
    }
} else {
    std::cout << "x <= 0" << std::endl;
}

// 陷阱2：赋值误写为比较
// if (x = 10) { ... }  // ❌ 赋值！不是比较！
if (x == 10) { ... }    // ✅ 比较
```

## 2. switch：高效的"跳转表"

### 生活类比

> switch 就像电梯的**楼层按钮**——直接跳到对应楼层（case），而不是逐层爬楼梯（else-if）。

### 概念讲解

```cpp
enum class Color { Red, Green, Blue };
Color c = Color::Green;

switch (c) {
    case Color::Red:
        std::cout << "红色" << std::endl;
        break;
    case Color::Green:
        std::cout << "绿色" << std::endl;
        break;
    case Color::Blue:
        std::cout << "蓝色" << std::endl;
        break;
}
```

### C++17 if with init + switch

```cpp
switch (auto status = 200; status) {
    case 200:
        std::cout << "OK" << std::endl;
        break;
    case 404:
        std::cout << "Not Found" << std::endl;
        break;
    default:
        std::cout << "Unknown status" << std::endl;
        break;
}
```

### switch vs else-if 对比

| 特性 | switch | else-if |
|------|--------|---------|
| 复杂度 | O(1) 跳转表 | O(n) 逐个判断 |
| 条件类型 | 必须是整数/枚举 | 任意可比较类型 |
| 代码简洁度 | 多分支更清晰 | 复杂条件更灵活 |

### 常见错误：忘记 break

```cpp
int v = 2;
switch (v) {
    case 1:
    case 2:
    case 3:
        std::cout << "v 是 1~3 之一" << std::endl;
        break;  // 只在最末尾 break
}

// C++17 显式声明 fall-through
switch (v) {
    case 1:
        std::cout << "one" << std::endl;
        [[fallthrough]];  // C++17 显式声明 fall-through
    case 2:
        std::cout << "fall-through 到这里" << std::endl;
        break;
}
```

## 3. for 循环：次数确定的"跑步圈"

### 生活类比

> for 循环就像跑道上的**圈数计数器**——有明确的起点、终点和步进规则。

### 概念讲解

```cpp
std::vector<int> nums = {1, 2, 3, 4, 5};

// 传统 for 循环
for (int i = 0; i < 5; ++i) {
    std::cout << nums[i] << std::endl;
}

// C++11 范围for（最常用）
for (int n : nums) {
    std::cout << n << std::endl;
}

// C++11 auto 简化范围for
for (auto& n : nums) {
    n *= 2;  // 引用可修改原数组
}
for (const auto& n : nums) {
    std::cout << n << std::endl;
}

// C++17 结构化绑定在循环中
std::vector<std::pair<std::string, int>> entries = {{"a", 1}, {"b", 2}};
for (const auto& [key, value] : entries) {
    std::cout << "key=" << key << ", value=" << value << std::endl;
}
```

### for 循环结构解析

```
for (①初始化; ②条件; ③步进)
       │          │         │
       ▼          ▼         ▼
    int i=0;   i < n;     ++i
       │          │         │
       └──────────┴─────────┘
              循环体
```

## 4. while 与 do-while：条件驱动的"循环门"

### 生活类比

> **while** 是在门口先看条件再进去（先判断后执行），**do-while** 是先刷卡再验证（先执行后判断，至少执行一次）。

### 概念讲解

```cpp
// while：先判断条件
int count = 0;
while (count < 3) {
    std::cout << "while count=" << count << std::endl;
    ++count;
}

// do-while：先执行，再判断（保证至少执行一次）
count = 0;
do {
    std::cout << "do-while count=" << count << std::endl;
    ++count;
} while (count < 3);
```

### 场景选择

| 类型 | 使用场景 |
|------|---------|
| while | 循环次数未知，条件可能一开始就不满足 |
| do-while | 用户输入验证（先读取，再验证） |

```cpp
// do-while 典型场景：用户输入验证
// do {
//     std::cout << "输入 1-10: ";
//     std::cin >> num;
// } while (num < 1 || num > 10);
```

## 5. break、continue、goto：跳转的"控制流"

### 生活类比

> - **break** 就像按电梯的**紧急停止**——立刻终止当前楼层所有活动
> - **continue** 就像跳过**不喜欢的节目**——继续看下一个
> - **goto** 就像传说中的**虫洞**——可以跳到任何地方（危险！）

### 概念讲解

```cpp
// break：跳出最近一层循环
for (int i = 0; i < 10; ++i) {
    if (i == 5) {
        std::cout << "遇到 5，跳出循环" << std::endl;
        break;  // 跳出整个 for 循环
    }
    std::cout << i << " ";
}
// 输出: 0 1 2 3 4 遇到 5，跳出循环

// continue：跳过本次迭代，进入下次
for (int i = 0; i < 8; ++i) {
    if (i % 2 == 1) {
        continue;  // 跳过奇数
    }
    std::cout << i << " ";  // 输出: 0 2 4 6
}
```

### goto 的合法使用场景

```cpp
// 场景1：跳出深层嵌套循环
for (int i = 0; i < 3; ++i) {
    for (int j = 0; j < 3; ++j) {
        for (int k = 0; k < 3; ++k) {
            if (k == 1) {
                std::cout << "找到目标，跳出全部循环" << std::endl;
                goto found_target;  // 唯一合理的 goto 用法
            }
        }
    }
}
found_target:  // goto 标签
    std::cout << "已到达目标位置" << std::endl;

// 场景2：函数早期退出（可以用 RAII 替代）
// void process() {
//     if (!init()) goto cleanup;
//     if (!load()) goto cleanup;
//     // ...
// cleanup:
//     release();
// }
```

### 最佳实践

> **break/continue 优先，goto 仅在必要时使用**（如跳出多层嵌套循环）。

## 6. 循环性能优化：让循环"跑得更快"

### 生活类比

> 循环优化就像整理房间的**动线规划**——把常用的东西放在手边，减少来回走动的时间。

### 常见优化技巧

```cpp
std::vector<int> data(1000000, 1);

// 陷阱1：在循环内重复计算长度
// 慢：
// for (int i = 0; i < data.size(); ++i) { ... }  // 每次都调用 size()

// 快：缓存 size()
int n = static_cast<int>(data.size());
for (int i = 0; i < n; ++i) {
    // data[i]...
}

// 陷阱2：循环展开（编译器通常自动做）
int sum = 0;
for (int i = 0; i < n; i += 4) {  // 每次处理4个
    sum += data[i] + data[i+1] + data[i+2] + data[i+3];
}
// 处理剩余元素
for (; i < n; ++i) {
    sum += data[i];
}
```

### 编译器优化级别

| 选项 | 说明 |
|------|------|
| -O0 | 无优化（调试用） |
| -O1 | 基本优化 |
| -O2 | 激进优化（常用） |
| -O3 | 最高优化（激进向量化） |

## 7. C++20 Ranges：更优雅的循环方式

### 概念讲解

```cpp
std::vector<int> nums = {5, 2, 8, 1, 9, 3};

// C++20 范围库（需要 -std=c++20）
// for (int n : nums | std::views::filter([](int x){ return x % 2 == 0; })
//                    | std::views::transform([](int x){ return x * 2; })) {
//     std::cout << n << " ";
// }

// C++17 等效实现
for (const auto& n : nums) {
    if (n % 2 == 0) {
        std::cout << n * 2 << " ";
    }
}
```

## 本章小结

| 结构 | 核心要点 |
|------|---------|
| if/else | 用括号明确块，避免悬空 else |
| switch | 跳转表 O(1)，忘记 break 会 fall-through |
| for | 范围 for + auto 是现代 C++ 最推荐的写法 |
| while | 先判断后执行，次数未知时用 |
| do-while | 先执行后判断，至少执行一次 |
| break | 跳出最近循环 |
| continue | 跳到下一次迭代 |
| goto | 仅在跳出深层嵌套时谨慎使用 |
