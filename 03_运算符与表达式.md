## 1. 算术运算符：数学运算的"工具箱"

### 生活类比

> 算术运算符就像厨房里的**基础工具**——刀（乘法）、砧板（除法）、余数收集盒（取模）。每种工具都有自己的使用规则。

### 概念讲解

| 运算符 | 名称 | 示例 |
|--------|------|------|
| + | 加法 | `17 + 4 = 21` |
| - | 减法 | `17 - 4 = 13` |
| * | 乘法 | `17 * 4 = 68` |
| / | 除法（整数除法向零取整） | `17 / 4 = 4` |
| % | 取模（余数） | `17 % 4 = 1` |

### 代码示例

```cpp
int a = 17, b = 4;

std::cout << "a + b = " << (a + b) << std::endl;  // 21
std::cout << "a - b = " << (a - b) << std::endl;  // 13
std::cout << "a * b = " << (a * b) << std::endl;  // 68
std::cout << "a / b = " << (a / b) << std::endl;  // 4（整数除法）
std::cout << "a % b = " << (a % b) << std::endl;  // 1（余数）

// 注意：除零是未定义行为！
// int c = 10 / 0;  // ❌ 运行时错误

// C++11 数值分隔符（C++14 起支持）
long long big = 1'000'000'000LL;

// 整数除法向零取整（C++11 规定）
std::cout << "-7 / 2 = " << (-7 / 2) << std::endl;  // -3（向零）
// 对比 Python：-7 // 2 = -4（向下取整）
```

### 常见错误

```cpp
// ❌ 整数除法丢失精度
int a = 5, b = 2;
double ratio = a / b;  // 结果是 2.0，不是 2.5！

// ✅ 正确做法：先转 double
double ratio = static_cast<double>(a) / b;  // 2.5
```

## 2. 赋值运算符：给变量"赋值"的艺术

### 生活类比

> `x += 5` 就像说"把 x 加上 5"，比 `x = x + 5` 更简洁——而且**复合赋值生成的机器码更少**。

### 概念讲解

| 运算符 | 等价于 | 说明 |
|--------|--------|------|
| += | x = x + y | 加法赋值 |
| -= | x = x - y | 减法赋值 |
| *= | x = x * y | 乘法赋值 |
| /= | x = x / y | 除法赋值 |
| %= | x = x % y | 取模赋值 |
| &= | x = x & y | 位与赋值 |
| \|= | x = x \| y | 位或赋值 |
| ^= | x = x ^ y | 位异或赋值 |

### 代码示例

```cpp
int x = 10;

x += 5;   // x = 15
x -= 3;   // x = 12
x *= 2;   // x = 24
x /= 4;   // x = 6
x %= 3;   // x = 0

// 位运算复合赋值
int flags = 0b1010;
flags &= 0b1100;  // flags = 0b1000
flags |= 0b0001;  // flags = 0b1001
flags ^= 0b0101;  // flags = 0b1100
```

### 性能优势

```
x = x + 5  生成的汇编:
  mov eax, [x]    ; 把 x 读入寄存器
  add eax, 5      ; 加 5
  mov [x], eax    ; 写回 x（3 条指令）

x += 5  生成的汇编:
  add dword ptr [x], 5  ; 直接加 5（1 条指令）
```

## 3. 关系运算符与逻辑运算符：判断的"逻辑门"

### 生活类比

> 逻辑运算符就像电路的**与门/或门/非门**：`&&` 是两个开关都闭合灯才亮，`||` 是任一开关闭合灯就亮，`!` 是开关反相。

### 概念讲解

**关系运算符**（返回 bool）：

| 运算符 | 含义 |
|--------|------|
| == | 等于 |
| != | 不等于 |
| < | 小于 |
| > | 大于 |
| <= | 小于等于 |
| >= | 大于等于 |

**逻辑运算符**：

| 运算符 | 含义 | 短路行为 |
|--------|------|---------|
| && | 逻辑与 | 左边为 false → 右边不执行 |
| \|\| | 逻辑或 | 左边为 true → 右边不执行 |
| ! | 逻辑非 | 取反 |

### 代码示例

```cpp
int a = 10, b = 20;

std::cout << "a == b → " << (a == b) << std::endl;  // 0 (false)
std::cout << "a != b → " << (a != b) << std::endl;  // 1 (true)

// 短路求值
bool cond1 = false, cond2 = true;
bool result = cond1 && cond2;  // cond2 不会被执行！
```

### 短路求值示例

```cpp
int divide_by_zero_called = 0;
bool short_circuit = false && (divide_by_zero_called++, true);
// 结果：divide_by_zero_called 仍然是 0（说明右侧未执行）
```

### C++20 spaceship operator

```cpp
auto result = a <=> b;
if (result < 0) std::cout << "a < b" << std::endl;
else if (result > 0) std::cout << "a > b" << std::endl;
else std::cout << "a == b" << std::endl;
```

## 4. 位运算：CPU 级别的"微观操作"

### 生活类比

> 位运算是拆开行李箱**逐个检查每件物品**的操作——不用整个箱子看，只看某个格子的状态。

### 概念讲解

位运算直接操作二进制位，是**最快的运算之一**（单 CPU 指令周期）。

### 常见位运算技巧

| 技巧 | 代码 | 说明 |
|------|------|------|
| 检查某位 | `flags & (1 << n)` | 判断第 n 位是否设置 |
| 设置某位 | `flags \| (1 << n)` | 将第 n 位设为 1 |
| 清除某位 | `flags &= ~(1 << n)` | 将第 n 位设为 0 |
| 翻转某位 | `flags ^= (1 << n)` | 0↔1 互换 |
| 判断奇偶 | `num & 1` | 奇数返回 1 |
| 交换两数 | `a ^= b; b ^= a; a ^= b;` | 不用临时变量 |

### 代码示例

```cpp
unsigned int flags = 0b11110000;

// 判断第3位是否设置
bool bit3_set = flags & (1 << 3);  // true

// 设置第2位
flags |= (1 << 2);  // 0b11110100

// 清除第3位
flags &= ~(1 << 3);  // 0b11110100（清除）

// 不用临时变量交换两数
int p = 5, q = 9;
p = p ^ q;
q = p ^ q;  // q = (p^q) ^ q = p
p = p ^ q;  // p = (p^q) ^ p = q
```

## 5. 运算符优先级：运算的"交通规则"

### 生活类比

> 优先级就像马路的**让行规则**——直行车让转弯车，乘法比加法优先级高（先乘除后加减）。

### 概念讲解

**C++ 优先级表（简化版，高优先级在前面）**：

```
1.  ::           作用域解析（最高）
2.  () [] . -> ++ --    后置操作
3.  ++ -- ! ~ + - * & sizeof   一元操作
4.  .* ->*      成员指针
5.  * / %       乘除取模
6.  + -         加减
7.  << >>       移位
8.  < <= > >=   关系
9.  == !=       相等
10. &            位与
11. ^            位异或
12. |            位或
13. &&           逻辑与
14. ||           逻辑或
15. ?:           条件（三目）
16. = += -= *= /= %=   赋值
17. ,            逗号（最低）
```

### 代码示例

```cpp
// 乘法优先于加法
int result = 2 + 3 * 4;      // = 14
int result2 = (2 + 3) * 4;   // = 20（括号改变）

// 结合性：同优先级从左到右
int assoc = 10 - 3 - 2;      // = (10-3)-2 = 5（左结合）
```

### 建议

> **不确定时用括号**——括号永远是最清晰的写法。

## 6. 三目运算符：表达式中的"if-else"

### 生活类比

> 三目运算符 `a ? b : c` 就像问路："路口左转吗？是的走这条路，不是走那条路"。一个问号一个冒号，一句话说完。

### 概念讲解

```cpp
// 基本用法
int min_val = (a < b) ? a : b;

// C++17 扩展：可以返回引用
int& smaller = (a < b) ? a : b;

// 嵌套用法（不推荐，优先用括号或 if-else）
char grade = (score >= 90) ? 'A'
           : (score >= 80) ? 'B'
           : (score >= 70) ? 'C'
           : (score >= 60) ? 'D' : 'F';
```

### 重要特性

> 三目运算符是**表达式**，不是**语句**——它可以返回值，可以用在初始化中。

```cpp
// ✅ 可以用在 const 初始化中
constexpr int max_val = (a > b) ? a : b;
```

## 7. constexpr 表达式：编译期的"数学家"

### 生活类比

> 普通表达式是"现场计算"，constexpr 表达式是"预约计算"——在编译阶段就完成计算，运行时直接用结果。

### 概念讲解

```cpp
// constexpr 函数（C++14 支持多语句）
constexpr int factorial(int n) {
    if (n <= 1) return 1;
    return n * factorial(n - 1);
}
constexpr int fact5 = factorial(5);  // 编译期计算：120

// constexpr lambda（C++17）
auto const_lambda = [](int x) constexpr { return x * x; };
constexpr int sq = const_lambda(12);

// C++17 constexpr if：编译期分支
// if constexpr (std::is_integral_v<T>) { ... }
```

## 本章小结

| 类别 | 核心要点 |
|------|---------|
| 算术运算 | 整数除法向零取整，注意除零错误 |
| 赋值运算 | 复合赋值（+= 等）比普通赋值效率更高 |
| 逻辑运算 | 短路求值避免无效计算 |
| 位运算 | 底层高效操作，常用于标志位、权限控制 |
| 优先级 | 不确定就用括号，括号不产生额外开销 |
| 三目运算 | 表达式而非语句，可用于 constexpr |
| constexpr | 编译期计算，运行时零成本 |
